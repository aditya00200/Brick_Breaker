# 🧱Brick Breaker

A classic **Brick Breaker** game made with **HTML, CSS, and JavaScript**. Bounce the ball with the paddle, break all the bricks, and try to beat your high score!  

---

## 🎮 Features

- **Multiple difficulty levels**: Easy, Medium, and Hard  
- **Score tracking** with high score saved in `localStorage`  
- **Responsive gameplay** inside a styled canvas  
- **Start screen & landing interface** with smooth transitions  
- **Restart game** option after Game Over or Game Win  
- **Interactive paddle and ball physics** for a fun challenge  

---

## 🕹 How to Play

1. Open `index.html` in your browser.  
2. Click **Enter Game** on the landing screen.  
3. Select a difficulty level: **Easy, Medium, Hard**.  
4. Use **Left** and **Right Arrow keys** to move the paddle.  
5. Bounce the ball to break all bricks and score points.  
6. Try to beat your **high score**!  
7. Click **Restart** or **Back** to start again.  

---

## ⚙️ Technologies Used

- **HTML5** – Structure of the game  
- **CSS3** – Styling, animations, and interface  
- **JavaScript** – Game logic, paddle & ball movement, collision detection, score handling
